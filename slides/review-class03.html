Importing Data from the cloud

Three options

  1. Regular R functions that take URLs

  2. API packages

    • RSocrata
    • tidyCensus
    • caladaptR

  3. Write your own download code


Socrata API

  • Used by a lot of orgs to share data (including DataSF)
  • Every dataset has a:
  • preview URL - tells you about the data, where it came from, which columns it has, preview, etc.
  • resource URL - direct link to the CSV, JSON, GeoJSON, or another standard filetype
  • Don’t need it all? Add a query parameter to the URL:

https://data.sfgov.org/resource/wwmu-gmzc.json?release_year=2011

  • RSocrata package
  • convenience functions
  • pass a API token
  • parse JSON results into a data frame
  • list datasets on a Socrata domain
  • write to a Socrata dataset
