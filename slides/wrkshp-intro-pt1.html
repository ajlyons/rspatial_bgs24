About You…

Workshop Goals

Get better at:

  • Working with R and RStudio
  • writing R commands
  • make mistakes
  • use functions
  • import tabular data
  • basic plotting

Learning Strategy

1) Understand foundational terms and concepts

2) Hands-on practice

3) Discover RStudio’s bells and whistles

4) Learn how to get help

Top Five Tips for Beginners

  1. Try to get working code
  2. Focus on adapting code rather than starting from scratch
  3. Review what we cover within 24 hours
  4. Ask for help for cryptic error messages
  5. Build up you own library of working code
