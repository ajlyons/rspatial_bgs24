Anatomy of a ggplot


Making maps with ggplot

Making ggplots with geospatial data (i.e., maps) is not that different than regular ggplots.

A few differences:

  • we use sf objects as the data source (must all be in the same CRS)
  • draw layers with geom_sf()
  • most layers will be based on different spatial data, so in general there is no need to define data in the opening ggplot()
  • order of the layers matters
  • the ggspatial package has functions you can use to add other map elements (e.g., scale bar)


geom_sf()

The magical powers of geom_sf():

  • no need to define the columns for ‘x’ and ‘y’ in aes(), geom_sf() figures it out!
  • geom_sf() will draw points, lines or polygons, depending on the feature type
  • Just like other geom functions, geom_sf() provides lots of options to customize the visual properties. Most of these can be mapped to attribute fields!
  • fill color
  • outline color
  • point size and shape
  • line color
  • etc.


Make a map of Yosemite Fire Points

library(sf)
epsg_utm11n_nad83 <- 26911
yose_bnd_utm <- sf::st_read(dsn="./data", layer="yose_boundary", quiet=TRUE) |> st_transform(epsg_utm11n_nad83)
ggplot() + geom_sf(data = yose_bnd_utm)


You may notice, no matter what the CRS of the underlying data, the graticules (gridlines) are drawn in long-lat.

To force the axes to use the projected map coordinates, add coord_sf().

ggplot() + geom_sf(data = yose_bnd_utm) + coord_sf(datum = st_crs(yose_bnd_utm))


Next, we add fire ignition points:

yose_firehist_pts_utm <- sf::st_read("./data/yose_firehistory.gdb", "YNP_FireHistoryPoints", quiet=TRUE)

ggplot() + 
  geom_sf(data = yose_bnd_utm) +
  geom_sf(data = yose_firehist_pts_utm)


Modify the visual properties:

ggplot() + 
  geom_sf(data = yose_bnd_utm, fill = "#5ab676") +
  geom_sf(data = yose_firehist_pts_utm, col = "red", cex =0.4, alpha = 0.2) +
  ggtitle("Fire Ignition Points in YNP")

Facet Maps

Here we add facet_grid() to create a separate plot for each fire cause (which is saved in the CAUSE column):

yose_firehist_pts_utm$CAUSE |> table()
## 
##   HC  LTG   MI 
##  829 3355  248
ggplot() + 
  geom_sf(data = yose_bnd_utm, fill = "#5ab676") +
  geom_sf(data = yose_firehist_pts_utm, col = "red", cex =0.4, alpha = 0.2) +
  facet_grid(. ~ CAUSE) +
  theme(axis.text = element_blank(), axis.ticks = element_blank()) +
  ggtitle(("Fire Ignition Points by Cause"))
